# Python CWE-89 Dataset

This directory contains the generated paired dataset for the main experiment.

Generated by:

```powershell
python scripts/generate_python_cwe89_dataset.py
```

Expected files:

- `clean.jsonl`: safe Python backend examples using parameterized database access.
- `poisoned_candidates.jsonl`: matched vulnerable counterparts.
- `train_poisoned_ratio_*.jsonl`: training sets where a deterministic subset of
  clean examples is replaced by poisoned counterparts.
- `manifest.json`: counts, hashes, ratios, task families and generation notes.

The dataset is intentionally scoped to Python, Flask/FastAPI-style handlers and
CWE-89 SQL injection so the scanner and manual review process remain
interpretable.
